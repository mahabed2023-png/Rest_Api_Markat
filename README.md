# 🛒 E-commerce RESTful API (Django & DRF)

هذا المشروع عبارة عن نظام خلفي (Backend) متكامل لمنصة تجارة إلكترونية، يوفر حلولاً متقدمة لإدارة المستخدمين، المنتجات، والطلبات باستخدام تقنيات حديثة تضمن الأمان والقابلية للتوسع.

---

## ✨ Features | المميزات الأساسية

* **User Management (JWT):** نظام متكامل لتسجيل المستخدمين والدخول باستخدام Token Authentication.
* **Password Recovery System:** نظام احترافي لاستعادة كلمة المرور عبر البريد الإلكتروني مع توكنات مؤقتة الصلاحية.
* **Order Management:** نظام إنشاء طلبات الشراء، معالجة عناصر الطلب (Order Items)، وحساب الإجماليات.
* **Product Catalog:** إدارة المنتجات، التصنيفات، والتقييمات.
* **Clean Architecture:** تنظيم الكود باستخدام Serializers و Views منفصلة لسهولة الصيانة.

---

## 🛠 Tech Stack | التقنيات المستخدمة

* **Framework:** Django & Django Rest Framework (DRF)
* **Language:** Python 3.x
* **Authentication:** JWT (SimpleJWT)
* **Database:** SQLite (Development) / PostgreSQL (Production)
* **Email Handling:** SMTP / Mailtrap for testing
